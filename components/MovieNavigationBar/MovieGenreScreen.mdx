# Movie Genre Screen

This screen displays all the genres. Cliking a genre will navigate the user to the selected movie genre screen.

## init()

---

Parameter: none

---

//

This function initiates the screen, requesting the data from the API and storing it in a variable to be rendered.

//

## renderGenres()

---

Parameter: none

---

//

This function takes the genres from the API call and renders them with a flatlist all genres in the flatlist contain a onGenrePress() function.

//

## onGenrePress()

---

Parameter: none

---

//

This function navigates the user to the selected genre movie screen. It also sends the data id and name to the selected genre movie screen.

//
